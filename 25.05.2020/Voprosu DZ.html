Вопросы для самоконтроля:

  1. Какая актуальная версия HTML, CSS, javascript

    2. HTML5, CSS3, Javascript ES10

  2. Классифицируйте теги (приведите минимум две классификации)

    3. Строчные, блочные, строчно-блочные, текстовые

  3. Что такое блочный элемент?

    4. Предназначен для создания основных частей веб-страницы, для разделения контента на блоки, занимает всю доступную ширину.

  4. Что такое строчный элемент?

    5. Является частью строки, используется для изменения текста, нельзя задавать ширину и высоту

  5. Допустимо ли вставлять блочный элемент в строчный?

    6. Нет

  6. Как из квадрата сделать круг?

    7. border-radius:50%

  7. Как отобразить сектор круга?

    8. div { width: 0px; height: 0px; border-radius: 50%; border: 50px solid green; border-top: 50px solid white; }

  8. Чем контейнер отличается от тега?

    9. Контейнер применяется для обертывания элементов

  9. Почему следует использовать спрайты?

    10. Для увеличения производительности и скорости загрузки веб-страницы

  10. Перечислите способы сделать элемент невидимым на странице.

    11. display:none, opacity: 0, background-color: transparent, закоментировать

  11. Какое позиционирование у контейнера html?

    12. static

  12. Перечислите способы назначения стилей.

    13. Через link, тег style, внутри тегов

  13. Почему у родителей элементов с position: absolute, float:left (при отсутствии другого контента) высота становится 0?

    14. Элементы выпадают из потока

  14. Перечислите все способы (какие знаете) отобразить div в строку.

    15. display:inline-block, через float, display: flex, display: grid, display: table-cell, через position.

  15. Почему у body по-умолчанию отступы слева-справа, сверху-снизу? Как их убрать?

    16. В браузерах есть встроенные стили по-умолчанию. Убрать: margin:0; padding:0.

  16. Какие типы селекторов вы знаете? (перечислить)

    17. по классу, тегу, id, по атрибуту, псевдоклассу

  17. Для чего нужны псевдоэлементы?

    18. Что бы добавить элемент, не изменяя исходный код

  18. Для чего нужны псевдоклассы?

    19. Что бы задать элементу стили, не изменяя исходный код

  19. Перечислите атрибуты тега img

    20. src, alt, title

  20. Сколько запросов одновременно может отправить браузер на получение изображений/стилей/файлов?

    21. Без ограничений

  21. Для чего нужно кеширование?

    22. Что бы увеличить скорость загрузки веб-страницы, за счет ранее сохраненных данных

  22. С помощью какого css-свойства можно изменить вид курсора?

    23. cursor

  23. С помощью какого свойства подключаются свои шрифты в css?

    24. @import, @font-face, тег link

  24. Чем отличается min-width от width? Что будет если min-width > width?

    25. min-width - минимальная ширина блочного элемента, при меньшей ширене экрана появится горизонтальный scroll. Если min-width > width, то ширина будет равной min-width.

  25. Что такое кросс-браузерность?

    26. Корректная загрузка веб-страницы на разных браузерах

  26. На какой вкладке dev-tool можно имитировать медленный интернет?

    27. network

  27. Что такое :hover?

    28. Отображение прописанного стиля для элемента при наведении на него курсора

  28. "Прочитайте" селектор как браузер: body>div>p:first-child:not(.store):not(#message)

    29. Применять для параграфа первого ребенка, который не является классом store с идентификатором message, который вложен в div, который вложен в body

  29. Какие плюсы в использовании иконочных шрифтов?

    30. Можно увеличивать без потери качества, большая скорость загрузки

  30. Чем em, отличается от ex?

    31. em - относительные ед. измерения. ex - высота символа.

  31. Что такое rem?

    32. rem - относительная ед. измерения, высота буквы М шрифта тега

  32. Где лучше располагать в html коде css?

    33. Распологать лучше в отдельном теге style или в отдельной файле с ссылкой на него.

  33. Как избежать "прыжков" контента при загрузке изображений?

    34. Необходимо указывать размеры изображения

  34. Что происходит, если указать только один из габаритов у изображения?

    35. Другое значени будет увеличиваться или уменьшаться пропорционально

  35. Как инициировать git-репозиторий?

    36. git init

  36. Как добавить файл на отслеживание?

    37. git add (имя файла), git add .

  37. Как создать контрольную точку git?

    38. git commit -m "..."

  38. Как отменить изменения в файле до последней контрольной точки в ветке?

    39. git checkout

  39. Чем git pull отличается от git push? А от git clone?

    40. git pull - получение данных из удаленного репозитория. git push - отправка данных на удаленный репозиторий. git clone - создание копии репозитория

  40. Что будет, если задать padding-left: -40px;

    41. Извенений не произойдет.

  41. Для чего использовать свойство title у HTML-элемента?

    42. title - название веб-страницы во вкладке браузера

  42. Как назначить фон третьему элементу в контейнере body?

    43. body>div:nth-child(3) {background-color;}

  43. Как сделать изображение на странице полупрозрачным

    44. opacity: 0.5
